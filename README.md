## Developers

- **Studio / Engine:** dev-server-tester, DioksidBright  
- **Client:** dev-server-tester  

## Included Libraries

- Qt 4.8.5
- Boost 1.56.0
- cURL 7.43.0
- OpenSSL 1.0.0
- SDL 2.0.4
- G3D

## Build Environment

- Visual Studio 2012 Ultimate

## Changes in v0.1

- Initial unofficial build of Roblox16 Studio.
- Legacy engine compiled successfully.
- Core functionality (Studio GUI, asset loading) working.
- Educational / test playgrounds included.
